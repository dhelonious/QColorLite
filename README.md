# QColorLite
Sublime Color Highlighter and Converter based on QColor


## Installation

### With Package Control

1. `Package Control: Add Repository` -> https://github.com/dhelonious/QColorLite
2. `Package Control: Install Package` -> `QColorLite`


## Settings
**phantom_shape**  - Shape of the color phantom displayed, "square" or "circle".<br/>
**hsl_precision**  - Number of decimal places in HSL precision.<br/>
**hex_upper_case** - Display hex values in upper case.<br/>
**named_colors**   - Enable phantoms for [named colors](https://www.december.com/html/spec/colorsvg.html)
                     such as "red", "blue".<br/>

### Menu Options
**QColorLite: Show Colors** - Show color phantoms.<br/>
**QColorLite: Hide Colors** - Hide color phantoms.<br/>
**QColorLite: Toggle Colors** - Toggle color phantoms.<br/>
**QColorLite: Convert to HEX** - Convert selected color into the HEX format.<br/>
**QColorLite: Convert to RGB** - Convert selected color into the RGB format.<br/>
**QColorLite: Convert to RGBA** - Convert selected color into the RGBA format.<br/>
**QColorLite: Convert to HSL** - Convert selected color into the HSL format.<br/>
**QColorLite: Convert to HSLA** - Convert selected color into the HSLA format.<br/>
**QColorLite: Settings** - Edit the QColor settings files.<br/>
